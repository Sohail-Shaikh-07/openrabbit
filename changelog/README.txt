OpenRabbit changelog archive
============================

This folder keeps plain-text release changelogs, one file per released
version. The canonical changelog remains CHANGELOG.md at the repository root.

Use these files when you need release notes in a simple text format for
GitHub releases, package metadata, announcements, or quick review.
